# cas-backend
cas-backend

## Prerequisite
- Java11 or above.
- maven 3.x.x or above.

## Usage
1. modify the [configuration file](src/main/resources/application.yml) according to your actual situation.
2. run `mvn package` in your terminal.
3. rum `java -jar project` in your terminal.
